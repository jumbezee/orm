2.Вводим хелп
3.Вводим dbn
4.Закрываем программу
5.Создалась папка migrate, если её еще не было и создался пустой файл с временной меткой в этой папке
6.Файл 1468938838_file.rb в папке migrate, переименовываем в 1468938838_create_cars.rb
  Где cars - это название создаваемой таблицы(можно написать любое название,но во мн.числе)

7.Запускаем приложение esterminal.rb
8.Вводим dbu - это обновит файл 1468938838_create_cars.rb
9.Добавляем нужные поля для таблицы в этом файле
10.Вводим dbc - это создаст таблицу в вашей БД

11.чтобы добавить строку, нужно в файле esterminal.rb, найти Create column и изменить условия создания
